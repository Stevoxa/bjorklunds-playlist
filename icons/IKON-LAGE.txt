UI-ikoner — raster (PNG från designblad) eller SVG-sprite

KÄLLA (designblad)
  icons/bild8/kalla/ikoner-och-grafiska-element.png

RASTER (standard i js/config.js: ICON_DISPLAY_MODE = 'raster')
  PNG ligger i icons/bild8/raster/*.png (en fil per ikon, t.ex. clipboard.png för #sym-clipboard).
  Uppdatera ur käll-PNG:  npm run icons:sheet
  Appen byter automatiskt <svg><use> mot <img> vid laddning — ingen query-parameter krävs.

TILLBAKA TILL SVG
  Sätt ICON_DISPLAY_MODE = 'svg' i js/config.js och ladda om.
  Alternativt (en gång per flik): sessionStorage.setItem('iconDisplayMode','svg'); location.reload()
  URL: ?icons=svg

ÖVRIGA URL-ÖVERSTYRNINGAR (?icons=…)
  raster | png | bild8  → PNG-läge
  external               → SVG + extern fil product-icons.svg
  inline                 → SVG + inline-sprite

BACKUP AV INLINE-SPRITE
  icons/ui-sprite-backup.svg
  Uppdateras med: npm run icons:extract
